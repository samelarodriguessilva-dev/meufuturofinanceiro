
# Meu Futuro Financeiro

App simples (React + Vite + TS) com 3 calculadoras:
- Aposentadoria INSS
- Previdência Privada (gráfico)
- Rescisão trabalhista

## Rodando localmente
```bash
npm install
npm run dev
```

## Deploy na Vercel
- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

## Estrutura
- `src/` código React
- `public/` manifest e ícones
